# ClaimsIQ v2.0 – Enterprise Intelligent Claims Platform

## Overview

ClaimsIQ v2.0 is a cloud-native, event-driven insurance claims processing platform built on AWS. The platform automatically ingests claims from multiple channels, processes various document formats, leverages Generative AI for claim analysis, applies business rules for decision-making, and orchestrates end-to-end workflows with enterprise-grade security, scalability, and observability.

## Key Features

- Multi-channel claim ingestion
- Intelligent document processing
- AI-powered claim analysis
- Automated decision engine
- Event-driven workflow orchestration
- Enterprise monitoring and security
- Infrastructure as Code using Terraform

## Technology Stack

- AWS
- Terraform
- Amazon Bedrock
- Amazon SES
- Amazon S3
- Amazon Textract
- AWS Lambda
- Amazon SQS
- AWS Step Functions
- Amazon DynamoDB
- Amazon CloudWatch
- Streamlit (Operations Dashboard)
